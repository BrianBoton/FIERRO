1 Panorama general

NotaAlcance del documento

Este informe desarrolla un estudio descriptivo sobre información asociada a la educación física. El análisis se organiza en seis bloques: caracterización de estudiantes, hábitos deportivos, tiempo de reacción, rendimiento físico, resistencia, lesiones y duración aeróbica.

En lugar de presentar solamente cálculos aislados, cada bloque incluye una tabla resumen, un gráfico y una lectura breve del resultado.

porc <- function(x, dec = 1) {
  paste0(round(x, dec), "%")
}

tabla_formateada <- function(x, encabezados = NULL, color = "#243B53") {
  tabla <- x

  if (!is.null(encabezados)) {
    names(tabla) <- encabezados
  }

  tabla %>%
    knitr::kable(align = "c") %>%
    kableExtra::kable_styling(
      full_width = FALSE,
      bootstrap_options = c("striped", "hover", "condensed", "responsive")
    ) %>%
    kableExtra::row_spec(0, bold = TRUE, color = "white", background = color)
}

frecuencias_ordenadas <- function(datos, variable) {
  datos %>%
    count({{ variable }}, name = "fi") %>%
    arrange({{ variable }}) %>%
    mutate(
      hi = fi / sum(fi),
      porcentaje = 100 * hi,
      Fi = cumsum(fi),
      Hi = cumsum(hi)
    )
}

percentil_lineal <- function(vector, p) {
  valores <- sort(vector)
  n <- length(valores)
  pos <- (p / 100) * (n + 1)
  entero <- floor(pos)
  decimal <- pos - entero

  if (entero < 1) return(valores[1])
  if (entero >= n) return(valores[n])

  valores[entero] + decimal * (valores[entero + 1] - valores[entero])
}

percentil_por_clase <- function(tabla, p,
                                f = "Cantidad_estudiantes",
                                acumulada = "Frecuencia_acumulada",
                                inferior = "Limite_inferior",
                                superior = "Limite_superior") {
  total <- sum(tabla[[f]])
  objetivo <- p / 100 * total
  fila <- which(tabla[[acumulada]] >= objetivo)[1]

  acumulada_anterior <- ifelse(
    fila == 1,
    0,
    tabla[[acumulada]][fila - 1]
  )

  L <- tabla[[inferior]][fila]
  fi <- tabla[[f]][fila]
  amplitud <- tabla[[superior]][fila] - tabla[[inferior]][fila]

  L + ((objetivo - acumulada_anterior) / fi) * amplitud
}

resumen_clases <- function(tabla, columna) {
  trabajo <- tabla %>%
    mutate(
      fi = .data[[columna]],
      Fi = cumsum(fi)
    )

  total <- sum(trabajo$fi)
  amplitud <- trabajo$limite_superior[1] - trabajo$limite_inferior[1]
  media <- sum(trabajo$marca * trabajo$fi) / total

  posicion_mediana <- total / 2
  clase_mediana <- which(trabajo$Fi >= posicion_mediana)[1]
  Fi_previa <- ifelse(clase_mediana == 1, 0, trabajo$Fi[clase_mediana - 1])

  mediana <- trabajo$limite_inferior[clase_mediana] +
    ((posicion_mediana - Fi_previa) / trabajo$fi[clase_mediana]) * amplitud

  clase_modal <- which.max(trabajo$fi)
  f_modal <- trabajo$fi[clase_modal]
  f_anterior <- ifelse(clase_modal == 1, 0, trabajo$fi[clase_modal - 1])
  f_siguiente <- ifelse(clase_modal == nrow(trabajo), 0, trabajo$fi[clase_modal + 1])

  moda <- trabajo$limite_inferior[clase_modal] +
    ((f_modal - f_anterior) /
       ((f_modal - f_anterior) + (f_modal - f_siguiente))) * amplitud

  varianza <- sum(trabajo$fi * (trabajo$marca - media)^2) / total
  desviacion <- sqrt(varianza)
  cv <- desviacion / media * 100

  tibble(
    media = media,
    mediana = mediana,
    moda = moda,
    varianza = varianza,
    desviacion = desviacion,
    coef_variacion = cv
  )
}

8 4. Rendimiento físico con simetría

8.1 Planteamiento

La distribución tiene siete intervalos de igual amplitud. Como es simétrica, las frecuencias opuestas son equivalentes:

n_1=n_7,\qquad n_2=n_6,\qquad n_3=n_5

Además, se conoce que H_6=0.95. Esto significa que el último intervalo representa el 5% del total. Como n_7=10:

10=0.05N \Rightarrow N=200

Luego, usando h_3=0.20:

n_3=0.20(200)=40

Y como n_2+n_5=48, con n_5=40:

n_2=8

Por simetría, n_6=8 y n_7=10.

12 Apoyo conceptual

12.1 Frecuencia relativa

h_i=\frac{f_i}{n}

12.2 Media para datos agrupados

\bar{x}=\frac{\sum f_iY_i}{n}

12.3 Coeficiente de variación

CV=\frac{s}{\bar{x}}\times 100

Un CV más bajo indica menor dispersión relativa y mayor homogeneidad.
